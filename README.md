## 📖 Installation Guide

### 🔧 **Prerequisites**
- Node.js v16 or higher
- Git installed
- WhatsApp account for bot

### 📱 **Method 1: Termux (Android)**

```bash
# Update packages
pkg update && pkg upgrade -y

# Install dependencies  
pkg install nodejs npm git -y

# Clone repository
git clone https://github.com/emmyhenz/EMMYHENZ-V1
cd emmyhenz-v1

# Install packages
npm install --no-bin-links

# Configure bot
cp config-example.js config.js
nano config.js

# Start bot
npm start
```

### 💻 **Method 2: Computer/VPS**

```bash
# Clone repository
git clone https://github.com/emmyhenz/EMMYHENZ-V1
cd emmyhenz-v1

# Install dependencies
npm install

# Configure settings
cp config-example.js config.js
nano config.js

# Run bot
npm start
```

### ⚙️ **Configuration**

Edit `config.js` with your details:

```javascript
module.exports = {
    WARN_COUNT: 3,
    APIs: global.APIs,
    APIKeys: global.APIKeys, 
    phoneNumber: "" // Don't Add If You Are Using Panel To Deploy
};
```
